### Lokaal draaien

Je kunt de website lokaal draaien door simpelweg `index.html` te openen in je browser, of gebruik een lokale webserver:

```bash
# Met Python 3
python3 -m http.server 8000

# Met Node.js (npx)
npx http-server
```

Bezoek dan `http://localhost:8000` in je browser.
